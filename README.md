# SoloCloser Legal Site (No Domain Needed)

This folder is a ready-to-deploy compliance website for App Store review.

## Included pages

- `index.html`: entry page
- `privacy-policy.html`: privacy policy (CN/EN)
- `terms-of-use.html`: terms of use (CN/EN)
- `support.html`: support and contact (CN/EN)

## 5-minute deploy with GitHub Pages (free)

1. Create a new public GitHub repository, for example: `solocloser-legal`.
2. Upload all files in this folder to the repo root.
3. In GitHub repo: `Settings -> Pages`.
4. Source: `Deploy from a branch`, choose `main` and `/ (root)`.
5. Wait 1-3 minutes, your site URL will be:
   - `https://<your-github-username>.github.io/solocloser-legal/`

## App Store Connect fields

- Privacy Policy URL:
  - `https://<your-github-username>.github.io/solocloser-legal/privacy-policy.html`
- Optional marketing/legal URL:
  - `https://<your-github-username>.github.io/solocloser-legal/`

## Must-edit items before publish

- Replace support email in:
  - `privacy-policy.html`
  - `terms-of-use.html`
  - `support.html`
- Optional: adjust pricing sentence to your final storefront strategy.
- Optional: update "Last updated" date.

## Notes

- No custom domain is required.
- HTTPS is provided automatically by GitHub Pages.
- Keep these URLs stable after submitting to App Store review.
